## 📞 Quando usar

### Se você é um expert/cliente novo:
1. Lê `00-variaveis-preenchidas.md` pra ver o **modelo Leo Tabari**
2. Compara com seu nicho · adapta tom de voz e tratamento plural
3. Volta no `template/mensageria-template.md` e substitui as variáveis pelos **seus** dados
4. Aprova templates Utility na Meta API com base em `template/whatsapp-templates-meta.md`

### Se você é da equipe Turbo Academy:
- É a fonte de verdade pra qualquer dúvida sobre **qual tag aplicar quando** · qual mensagem disparar pra qual tier · qual horário de cada disparo
