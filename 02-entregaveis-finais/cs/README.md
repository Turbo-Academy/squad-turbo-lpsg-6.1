## 🎯 Função estratégica

CS é a **última fronteira do funil**. Sem CS, todo o resto vira receita única que precisa ser refeita do zero a cada lançamento.

Com CS bem estruturado:
- **Reduz churn** (D7-D30-D60)
- **Coleta prova social** (alimenta próximas vendas)
- **Gera receita extra** (ascensão + indicação)
- **Cria comunidade** que se vende sozinha
